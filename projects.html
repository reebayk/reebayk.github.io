🏨 Multithreaded Hotel Reservation System

A thread-safe hotel reservation system built in C using POSIX Threads. Went through multiple debugging passes to land on a single mutex protecting all shared state, so an auditor always reads a fully consistent snapshot — no overbooking, no race conditions, verified under stress testing.
